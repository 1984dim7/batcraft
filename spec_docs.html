Batios Physics Core

ゲーム物理演算＆パラメータ仕様

『Batios』の削り出しバット性能計算、打撃反射、飛距離マルチプライヤー、および球種とスイングタイミングによる発光システムの完全解説書。

1. バット形状とステータス計算

削られたバットの形状（長さ 84 cm、80個のセグメント半径 radii）から、以下の物理パラメータが実時間で計算されます。

M = Σ dm    (dm = ρ · π · r² · dx)
各セグメントの半径 r と木材密度 ρ (750 kg/m³) から総質量 M (kg) を算出します。太く残すほど重くなります。
com = Σ (dm · x) / M
グリップ（0m）からの重心位置 com (m) を算出。先端が太いほど重心は前方（ヘッド側）に寄ります。
I = Σ (dm · x²)
慣性モーメント I (kg·m²)。バットの振り抜きにくさを表し、重心が遠く、重いほど巨大になります。
cop = I / (M · com)
打撃の中心 (Center of Percussion)。バットを振った際に手に最も衝撃が来ない、ゲーム内の「芯（スイートスポット）」の位置です。重心 com よりもややヘッド寄りに形成されます。

これらの基礎物理値から、ゲーム上の実ステータスが決定されます。

  • スイング時間 (swingT): 慣性モーメント I に比例して長くなります。swingT = clamp(0.16 + 0.16 * I, 0.19, 0.75)。重くヘッド寄りのバットほど、タップしてからボールを捉えるまでに時間がかかります。
  • 芯の太さ (barrel): バットの先端側（60%以降）の最大半径。クリーンヒット判定の許容誤差を広げます。
  • 芯での有効質量 (meff): meff = I / (cop * cop)。衝突時にボールに伝えられるバットの実行力です。
  • しなりやすさ (flex): グリップ周辺（5%〜40%）の最小半径 handleR に反比例します。細いバットほど強烈にしなります。
2. 打撃反射と飛距離計算

バットとボールの衝突が検知されると、精密な跳ね返り物理（反発係数 $eA$）と、ゲーム性を高める飛距離補正が行われます。

q = exp( -((x - cop) / 0.14)² )
ミートの質 q。バット上の衝突位置 x が、芯 cop からズレるほど指数関数的（ガウス分布）に反発力が減衰します。
eA = clamp( (0.05 + 0.50 · q - rr) / (1 + rr), -0.5, 0.6 )    (rr = M_ball / meff)
反発係数 eA。ボール質量 (0.145kg) とバットの有効質量 meff の比率 rr、およびミートの質 q から算出。重いバットで芯に当てるほど反発係数が高まり（最大 0.6）、詰まると押し負けます。

衝突時の反発速度 $vOut$ は、バット表面の法線ベクトル $n$ を用いた反射ベクトルに、バットスイング速度 $vBat$、ボール速度 $vBall$ を加味して計算されます。

E = q · (vBat / 22) · (meff / 3)   ⇒   mult = clamp( exp((E - 0.82) · 3.2), 1.0, 12.0 )
飛距離マルチプライヤー (mult)。芯で完璧に捉え、スイング速度が速い（鋭い振り）ほど、飛距離が最大 12倍 まで跳ね上がります。

さらに、飛翔中の空気抵抗（ドラッグ）は dragK = 0.0042 / mult として計算されます。つまり、完璧に芯で捉えた打球ほど空気抵抗が弾丸のように激減し、減速せずに遥か場外まで飛んでいく仕様となっています。

3. タイミングとボール発光色

投球されたボールは最初は白色（0xffffff）で飛んできます。スイングしてボールを捉えた瞬間、タイミングの正確さに応じてボールそのものと周囲の光の輪（ハロー）が以下の通り変化します。

タイミング 比率 t 発光色 解説
Early (早い) 0.36 ～ 1.0 青 ～ 緑 バットの出が早い。引っ張る方向へ飛びやすい。
Perfect (ジャスト) 0.32 ～ 0.36 黄色 (Yellow) 最もバット速度が速く、芯でミートできる最高の瞬間。
Late (遅い) 0.0 ～ 0.32 橙 ～ 赤 引きつけすぎ、または振り遅れ。流し方向へ飛びやすい。
当たる判定外 (空振り) - 白色 (White) タイミングが完全にズレているため、色は変わらず白のまま空振ります。

※ ジャストタイミング（黄色付近）かつ芯で捉えた大飛球（ジャストミート）の際は、火花エフェクトにも純白や電光水色がブレンドされ、強烈な閃光が走ります。

4. 素材別パラメータ一覧

削り出すマテリアル（素材）によって、性能倍率（パワー、しなり、振りやすさ）が異なります。

素材 パワー倍率 しなり倍率 スイング時間倍率 特徴
木製 1.00 1.00 1.00 バランスが良く、最もスタンダードな素材。
カーボン 0.95 0.55 0.72 (速い!) 非常に軽量でスイング時間が短く、振り抜きやすい。
アルミ 1.25 0.30 0.95 硬くて高反発。金属バット特有の打球速度が出る。
金 1.40 0.15 1.25 (重い) 金密度が高いため重いが、衝突時の破壊力は絶大。
ダイヤ 1.60 (最強) 0.05 1.10 (重い) 極めて硬くしならないが、最長飛距離を狙えるロマン素材。
黒曜石 1.30 0.10 1.05 高反発かつ、やや重量のある鋭い打球を生む。
