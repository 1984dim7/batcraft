0. レーダーステータス 5項目の意味

レーダーチャートの各頂点が何を計算しているか、ゲームプレイにどう影響するかを解説します。

⚡ パワー
「有効質量 meff」に比例。
バットを振ったとき、ボールとの衝突で実際に力を伝えられる質量。
大きいほど反発係数が上がり、飛距離マルチプライヤーが伸びる。
power = clamp(meff / 3.5, 0.06, 1) × 100
🎮 操作性
「スイング時間 swingT（秒）」の短さ。
タップしてからボールを捉えるまでの時間。
遅いほど「早押し」が必要になり、タイミングがシビア。
handling = clamp((0.8 − swingT) / 0.55, 0.06, 1) × 100
swingT自体は慣性モーメントIと素材倍率で決まる。
🌿 しなり
グリップ最細部の細さから計算されるムチ効果。
細いグリップほど、インパクト時にバットがしなって
バット先端速度を上乗せする（meff・flexを参照）。
flex = (0.016/handleR)³ × (1 + I×2) × flexMult
🛡 耐久性
バットのHPの最大値。
1球ごとにHPが減り（4〜12 dmg）、0になるとバット破損でゲーム終了。
形状の均整度・最薄部・素材倍率で決まる。
均一な太さ ＆ 細くしすぎない → 高耐久。
🎯 ミート
「芯ゾーンの広さ」。
バットの60〜95%区間で最大半径の80%以上を維持しているセグメントの割合×バレル半径。
大きいほど「少し詰まっても飛ぶ」確率が上がる。
meet = barrelWidth × barrel × 2500 × meetMult
💡 操作性 ≠ ミート
操作性はスイング所要時間（いつ押すか）のシビアさ。ミートは芯の広さ（どこに当てるか）の許容度。 重くて遅いバット（低操作性）でも芯が広ければ当てやすい。軽くて速くても芯が狭ければ難しい。両方独立している。
1. バット形状 → ステータス計算の全式

ゲーム内の定数（index.html と完全一致）

定数値意味
SEG56バット分割セグメント数
BAT_LEN0.85 mバット全長
LOG_R0.062 m丸太初期半径（6.2 cm）
MIN_R0.011 m削れる最小半径（1.1 cm）
RHO700 kg/m³木材密度（実際の木と同等）
MB0.145 kgボール質量
BALL_R0.037 mボール半径
BAT_BOOST1.9体重移動ぶんのバット速度補正係数

セグメント積分（i=0 がグリップ端、i=SEG がヘッド端）

dx = BAT_LEN / SEG = 0.85/56 ≈ 15.2 mm/セグメント
dm_i = RHO × π × r_i² × dx
M = Σ dm_i （総質量）
com = Σ(dm_i × x_i) / M （重心。グリップから何m）
I = Σ(dm_i × x_i²) （慣性モーメント kg·m²）
cop = I / (M × com) （打撃の中心 = スイートスポット位置）
x_i = i × dx。0がグリップ端、0.85mがヘッド端。

各性能値への変換

swingT = clamp( (0.15 + 0.20 × I) × swingTMult, 0.18, 0.80 ) [秒]
Iが大きい（重くヘッド寄り）ほど遅くなる。素材倍率が掛かる。
meff = I / (cop² + 1e-4) × powerMult × shapeScore
衝突時に実際に機能する有効質量。powerMultは素材倍率。shapeScoreはバット形状の良さ（0.1〜1.0）。
handleR = min(r_i) for i in [5%, 40%] of SEG
flex = clamp( (0.016 / handleR)³ × (1 + I×2) × flexMult, 0.2, 3.5 ) × shapeScore
グリップ最薄部が細いほどしなる。I が大きいほど加速する。
barrel = max(r_i) for i > SEG×0.6
barrelWidth = (r_i ≥ maxBarrelR×0.8 な i の数) / (SEG×0.5 to SEG×0.95 の区間長)
meet = clamp( barrelWidth × barrel × 2500 × meetMult × shapeScore, 5, 100 )
2. 打撃反射と飛距離マルチプライヤー
q = exp( −((x_contact − cop) / 0.14)² )
接触位置が芯(cop)からズレるほど指数的に減衰するミートの質（0〜1）。
rr = MB / (I / (x_contact² + 1e-4))
eA = clamp( (0.05 + 0.50×q − rr) / (1 + rr), −0.5, 0.6 )
反発係数 eA。芯(q=1)で重いバット(meff大)ほど高まり、詰まると押し負ける。
E = q × clamp(vBat/22, 0, 1.3) × clamp(meff/3, 0.5, 1.2)
mult = min( exp( max(0, E − 0.82) × 3.2 ), 12.0 )
飛距離マルチプライヤー（SACRED CONSTRAINT: 変更禁止）。
E < 0.82 では mult=1（増幅なし）。E=1.0で約2.5倍、E=1.56で最大12倍。
dragK = 0.0042 / mult。完璧なヒットほど空気抵抗が激減する。
💡 「操作性を下げて代わりに何を得るか」のトレードオフ
重いバット（I大）→ swingTが長い → 操作性低下。
しかし重いバット → meffが大きくなりやすい → E の第3因子 clamp(meff/3, 0.5, 1.2) が上がる → mult増加 → 飛距離増。
ただし meff/3 ≥ 1.2 でキャップされるので、それ以上重くしても飛距離は増えない。「重くて遅いのに飛ばない」地帯が存在する。
3. スイングタイミングとボール発光

タップからボールを捉えるまでの時間は leadT × swingT = 0.40 × swingT 秒。
つまりswingT=0.35秒なら 0.14秒前に押す必要がある。swingT=0.65秒なら0.26秒前。
