# 共有ランキングのセットアップ手順（約5分）

ランキングは Google スプレッドシートに保存され、Google Apps Script (GAS) のWebアプリ経由で読み書きされる。プレイヤー側のログインは不要。

## 1. スプレッドシートを作る

1. https://sheets.new を開く（dim7rai@gmail.com でログインした状態で）
2. ファイル名を「バット工房ランキング」などに変更

## 2. Apps Script を貼り付ける

1. メニューの **拡張機能 → Apps Script** を開く
2. エディタに最初から入っているコードを全部消して、`ranking-backend.gs` の中身を貼り付ける
3. 保存（Cmd+S）

## 3. Webアプリとしてデプロイ

1. 右上の **デプロイ → 新しいデプロイ**
2. 歯車アイコン → 種類で **ウェブアプリ** を選択
3. 設定:
   - 次のユーザーとして実行: **自分**
   - アクセスできるユーザー: **全員**
4. **デプロイ** を押す（初回は権限の承認画面が出るので許可する）
5. 表示された **ウェブアプリのURL**（`https://script.google.com/macros/s/.../exec`）をコピー

## 4. ゲームにURLを設定

`index.html` 内の

```js
const RANK_API = '';
```

をコピーしたURLに書き換える:

```js
const RANK_API = 'https://script.google.com/macros/s/XXXX/exec';
```

（ClaudeにURLを渡せば設定からプッシュまでやります）

## 動作の仕組み

- **登録**: ゲーム終了画面の「ランキングに登録」で `{name, total, mat, date}` がPOSTされ、シートに1行追記される
- **表示**: ランキング欄はまずローカル記録を即表示し、APIから共有ランキングを取得でき次第差し替える（オフラインでもローカル表示で動く）
- **素材併記**: 名前の横に（木製）（カーボン）（金）などが表示される
- 不正値（負の距離・1000万m超え等）はGAS側で破棄される

## メンテナンス

- 記録の削除・編集はスプレッドシートを直接編集すればよい
- スクリプトを修正したら「デプロイ → デプロイを管理 → 編集 → 新バージョン」で更新する（URLは変わらない）
